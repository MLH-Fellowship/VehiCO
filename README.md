# VehiCO

## Getting Started

### Setup

- Fork and clone the repo

```
$ git clone https://github.com/MLH-Fellowship/VehiCO.git
$ cd VehiCO
```

#### Frontend:
- Install dependencies
```
$ cd client
$ npm install
```

- Run the server and react app

```
$ npm run dev
```

#### Backend:
- Install dependencies
```
$ cd server
$ npm install
```

- Run the server
```
$ npm start
```
